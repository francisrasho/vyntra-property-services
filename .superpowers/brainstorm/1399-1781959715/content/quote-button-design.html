How it works technically

Claude includes a hidden marker [SHOW_QUOTE_BUTTON] at the end of its response when relevant. The frontend strips the marker and renders the real button — which opens the existing Quote Modal already on the site.

When does Claude show the button?

  • User asks about pricing or cost
  • User describes a specific job or scope
  • User asks how to book or get started
  • User says they're ready to proceed
